# my-Server Modpack

Minecraft 1.21.1 modpack with Tech, Magic, and RPG archetypes.

## 🎮 Установка (Для ЛИЦЕНЗИИ)

1. Скачай **Prism Launcher**: https://prismlauncher.org/
2. Открой Prism Launcher
3. Нажми **Add Instance** → **Import from URL**
4. Вставь эту ссылку:
   ```
   https://github.com/Fionkor/my-Server
   ```
5. Нажми **Install**
6. Нажми **Play** ✅

## 🏴‍☠️ Установка (Для ПИРАТКИ)

### Способ 1: PolyMC (Рекомендуется)
1. Скачай **Prism Launcher**: https://prismlauncher.org/
2. Повтори шаги выше (работает для всех)

### Способ 2: MultiMC
1. Скачай **MultiMC**: https://multimc.org/
2. Import modpack (выбери ZIP архив)
3. Запусти

### Способ 3: Ручная установка
1. Клонируй или скачай репо
2. Скопируй папки `mods` и `config`
3. Убедись что стоит **NeoForge 21.1.233**
4. Запусти игру

## 📝 Характеристики

- **Minecraft**: 1.21.1
- **Forge**: NeoForge 21.1.233
- **Модов**: 69
- **Архетипы**: Tech, Magic, RPG
